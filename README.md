# OpenWebUI_chat_metrics-
Chat Metrics for Open WebUI

Based on https://openwebui.com/f/iamg30/cogito 

Changes:
  - Count token total per Chat instead of per Session
  - updated Models

How to use:

In OpenWebUI -> Admin Panel -> Functions -> 
either a) New Function -> New Function -> replace example code with contents of chat_metrics.json 
or b) (if you downloaded chat_metrics.json) Import 

Enable function in Functions overview (slider), then click ... and enable Global

Click on cog wheel to change which stats are shown.
